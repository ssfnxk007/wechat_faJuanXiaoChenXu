# 迁移与实体时间字段约定

**适用范围：** 2026-04-20 起的所有**新**迁移与**新**实体。
**不追溯：** 历史表保留 `datetime` 现状；如需统一迁移，走审核项 #22 单独子计划。

## 1. SQL DDL 列类型

| 场景 | 规范 | 示例 |
|---|---|---|
| 时间戳列（CreatedAt / UpdatedAt / 业务时刻） | `datetime2(3)` | `[CreatedAt] datetime2(3) NOT NULL CONSTRAINT [DF_Foo_CreatedAt] DEFAULT (SYSDATETIME())` |
| 日期列（生日、有效期仅日粒度） | `date` | `[BirthDate] date NULL` |
| 并发控制列 | `rowversion` | `[RowVersion] rowversion NOT NULL` |

**为什么 `datetime2(3)`：**
- `datetime` 精度 3.33ms、占 8 字节；`datetime2(3)` 精度 1ms、占 7 字节，存储更紧且精度更高
- `datetime2` 范围 `0001-01-01 ~ 9999-12-31`，`datetime` 仅 `1753-01-01 ~ 9999-12-31`
- SQL Server 官方推荐新项目使用 `datetime2`

**为什么不强推 `datetimeoffset`：**
- 项目业务锁定单一时区（Asia/Shanghai），不需要随列带时区；`datetime2` 已足够，写入开销更低

## 2. EF Core 实体约定

**所有业务实体**（领域对象，不含纯关联表、纯枚举映射）必须实现：

```csharp
public interface IAuditable
{
    DateTime CreatedAt { get; set; }
    DateTime UpdatedAt { get; set; }
}
```

- `CreatedAt`：实体首次 `INSERT` 时赋值，后续不可变
- `UpdatedAt`：每次 `UPDATE` 时由 `SaveChangesInterceptor` 刷新
- 两列在实体定义里**不**给默认值（`= DateTime.Now`），由拦截器负责

**纯关联表**（如 `AdminUserRole`、`AdminRoleMenu`）可省略 `UpdatedAt`，只保留 `CreatedAt`。

## 3. SaveChangesInterceptor 模式（后续 #22 落地）

```csharp
public sealed class AuditingInterceptor : SaveChangesInterceptor
{
    public override InterceptionResult<int> SavingChanges(
        DbContextEventData eventData, InterceptionResult<int> result)
    {
        var now = DateTime.Now;
        foreach (var entry in eventData.Context!.ChangeTracker.Entries<IAuditable>())
        {
            if (entry.State == EntityState.Added)
                entry.Entity.CreatedAt = now;
            if (entry.State == EntityState.Added || entry.State == EntityState.Modified)
                entry.Entity.UpdatedAt = now;
        }
        return base.SavingChanges(eventData, result);
    }
}
```

项目 DbContext 注册：`options.AddInterceptors(new AuditingInterceptor())`。

## 4. 迁移脚本模板

```sql
-- NNN-<topic>.sql
BEGIN TRANSACTION;
GO

IF NOT EXISTS (SELECT 1 FROM sys.tables WHERE name = N'Foo')
BEGIN
    CREATE TABLE [dbo].[Foo]
    (
        [Id] bigint IDENTITY(1,1) NOT NULL,
        -- 业务列...
        [CreatedAt] datetime2(3) NOT NULL CONSTRAINT [DF_Foo_CreatedAt] DEFAULT (SYSDATETIME()),
        [UpdatedAt] datetime2(3) NOT NULL CONSTRAINT [DF_Foo_UpdatedAt] DEFAULT (SYSDATETIME()),
        CONSTRAINT [PK_Foo] PRIMARY KEY CLUSTERED ([Id] ASC)
    );
END;
GO

COMMIT;
GO
```
