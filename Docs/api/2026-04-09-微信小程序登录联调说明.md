# 微信小程序登录联调说明

## 后端接口

- 配置状态检查：`GET /api/auth/wechat-status`
- 小程序登录：`POST /api/auth/mini-login`
- 手机号绑定：`POST /api/auth/bind-mobile`

## 配置项

请在以下文件填写真实配置：

- `backend/src/FaJuan.Api/appsettings.json`
- 或 `backend/src/FaJuan.Api/appsettings.Development.json`

需要填写：

```json
"WeChatMiniProgram": {
  "AppId": "wx...",
  "AppSecret": "..."
}
```

## 联调步骤

1. 启动后端 API
2. 调用 `GET /api/auth/wechat-status`
3. 若返回 `IsConfigured = true`，表示后端已具备真实微信登录条件
4. 小程序端调用 `wx.login()` 获取 `code`
5. 将 `code` 提交到 `POST /api/auth/mini-login`
6. 后端通过微信 `code2Session` 获取：
   - `openid`
   - `session_key`
   - `unionid`（如微信返回）
7. 系统根据 `openid` 自动建档或更新用户

## 请求示例

```json
{
  "code": "wx-login-code",
  "nickname": "测试用户"
}
```

## 返回示例

```json
{
  "code": 200,
  "message": "success",
  "data": {
    "userId": 1,
    "miniOpenId": "oXXXXXXXX",
    "mobile": null,
    "isNewUser": true
  },
  "success": true
}
```

## 注意事项

- `code` 只能使用一次，且有效期较短
- 若未配置 `AppId/AppSecret`，后端会返回明确错误信息
- 当前小程序前端尚未正式接入 `wx.login()` 页面逻辑；后续做 `miniapp/` 时直接对接此接口即可
