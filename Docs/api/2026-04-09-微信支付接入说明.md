# 微信支付接入说明

## 后端接口

- 配置状态检查：`GET /api/payments/wechat-status`
- 创建支付单：`POST /api/payments/create`
- 真实支付回调：`POST /api/payments/callback`
- 模拟支付回调：`POST /api/payments/callback/mock`

## 必填配置

在 `backend/src/FaJuan.Api/appsettings.Development.json` 或生产配置中填写：

```json
"WeChatPay": {
  "AppId": "wx...",
  "MerchantId": "...",
  "MerchantSerialNo": "...",
  "PrivateKeyPemPath": "C:/keys/apiclient_key.pem",
  "ApiV3Key": "...",
  "NotifyUrl": "https://your-domain/api/payments/callback",
  "EnableMockFallback": true
}
```

## 说明

- `AppId`：小程序 AppId
- `MerchantId`：微信支付商户号
- `MerchantSerialNo`：商户 API 证书序列号
- `PrivateKeyPemPath`：商户私钥 PEM 文件绝对路径
- `ApiV3Key`：微信支付 APIv3 密钥
- `NotifyUrl`：微信支付回调地址
- `EnableMockFallback`：未配置完成时是否允许回退到模拟支付

## 当前实现范围

- 已接入 JSAPI 下单请求签名
