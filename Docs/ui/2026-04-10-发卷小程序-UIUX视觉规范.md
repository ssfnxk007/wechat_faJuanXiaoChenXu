# 发卷小程序 (Ultra Vision) UI/UX 视觉规范与前端落地指南

> **文档定位**：本规范基于 `Docs/ui/miniapp-preview-v1.html` 终极版（Ultra Vision V2）提取，作为后续 UniApp 开发过程中的**像素级 CSS 与组件复刻红皮书**。严格禁止偏离此规范开发。

## 1. 核心设计语言 (Core Design Language)

本次彻底抛弃传统的扁平化和小程序原生呆板样式，全面升级为类原生 App 的**玻璃拟态 (Glassmorphism)** 与 **高质感悬浮 (Floating Layer)** 设计。

### 1.1 绝对禁令 🚫
- **禁止使用 Emoji 表情**：所有图标强制使用高质量内联单色/双色 SVG 解析，杜绝平台渲染差异。
- **避免纯高饱和色平铺**：大面积色彩必须采用渐变 (Gradient) 或配合毛玻璃 (Backdrop-filter) 使用。

### 1.2 色彩阶梯 (Color Palette)

*   **主品牌色 (Primary)**:
    *   主极光色：`#FA541C` (火山红橙)
    *   柔金辅助：`#FF9C6E`
    *   **核心品牌渐变 (Button & Banner)**: `linear-gradient(135deg, #FF6A00 0%, #EE0979 100%)`
    *   发光阴影色：`rgba(250, 84, 28, 0.15)`
    *   极弱背景 (辅助Tag): `#FFF2E8`

*   **辅助态 (Status)**:
    *   成功 (券可用): `#10B981` (背景: `#D1FAE5`)
    *   暗色大背景 (用于沉浸模式): `#2D3748`

*   **文字阶梯 (Typography Color)**:
    *   核心大标题/强推元素：`#111827`
    *   正文：`#4B5563`
    *   弱化文字 (时间、描述): `#9CA3AF`

## 2. 交互与动效要求 (Interactions & Animations)

在编写 UniApp Vue SFC 组件时，必须要实现的反馈动画：

1.  **按压反馈 (Press Feedback)**:
    *   所有按钮元素必须拥有专属 `:active` 伪类或 `hover-class`。
    *   规范缩放比：`transform: scale(0.94)`
    *   规范过渡：`transition: all 0.2s;`
2.  **页面切换 (Page Transitions)**:
    *   采用平滑的 Fade-Up 进场：`transform: translateY(16px)` 到 ` translateY(0)`。
3.  **沉浸式状态转换 (Immersive Switch)**:
    *   如果页面带深色头图（如：**首页轮播图**、**个人中心**），状态栏字体**强制白色**，导航栏完全透明。
    *   如果在纯浅色页面（如：**商城**、**我的券**），状态栏强制黑色，导航栏开启**高级模糊 `backdrop-filter: blur(12px)`**。

## 3. 核心关键组件实现 (Core Components Implementation)

### 3.1 Swiper 轮播宇宙
- 强制使用**边缘内缩边距**，不能傻呆呆地顶格四边。
- `border-bottom-left-radius` 统一设定为 36px 级大圆角。
- 当背景颜色过浅时，需要有逻辑监听并触发状态栏字体反黑。

### 3.2 Floating TabBar 悬浮胶囊导航
- **定位**：`bottom: 20px; left: 16px; right: 16px;`，悬空设计。
- **质感**：背景 `rgba(255,255,255,0.9)` + 毛玻璃 `backdrop-filter: blur(24px)` + 顶部极细白边反光 `inset 0 1px 0 rgba(255,255,255,0.6)`。
- **点击状态**：
    1.  原图标被填充一个主色的圆形底色，图标变色。
    2.  `transform: translateY(-4px);` 并带着光环阴影。
    3.  底部原本的“文字说明”（如：主页）迅速 FadeOut 并缩小。

### 3.3 物理质感砍刀缝（卡券专用）
优惠券中间的物理分割线，需使用伪元素构建圆形内凹：
- 顶部内凹：`top: -12px; width: 24px; height: 24px; background: 页面背景层; border-radius: 50%;`
- 底部同理，并在两者中间拉一条 `2px dashed #E5E7EB`。
