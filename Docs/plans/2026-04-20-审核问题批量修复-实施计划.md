## 决策记录：#11 全项目 DateTime.Now → UtcNow **不做**

**决策依据：**
- 业务为国内单一时区场景（发券小程序）；用户、商户、核销门店均在 Asia/Shanghai
- 项目未容器化（无 Dockerfile），服务器本地 TZ 固定后稳定
- 34 个后端文件、58 处 `DateTime.Now`；DB 大量 `datetime` 列已按本地时间落盘；前端/miniapp 全部按本地时间直显
- 改造涉及历史数据迁移 + 前端显示层转换，成本高、回归风险大、收益边际

**兜底措施：** 以 Task 1（TZ 断言）替代，fail fast 胜于全量改造。

```csharp
public async Task<(bool, string)> MarkOrderPaidAsync(...)
{
    if (transaction.Status == PaymentStatus.Success) return (true, "支付已处理");

    using var tx = await dbContext.Database.BeginTransactionAsync();
    try
    {
        // ... 现有逻辑
        await dbContext.SaveChangesAsync();
        await tx.CommitAsync();
        return (true, "支付成功并已发券");
    }
    catch (DbUpdateConcurrencyException)
    {
        await tx.RollbackAsync();
        // 并发另一侧已提交；回查一次确认最终态
        var current = await dbContext.PaymentTransactions.AsNoTracking()
            .FirstAsync(x => x.Id == transaction.Id);
        return current.Status == PaymentStatus.Success
            ? (true, "支付已处理（并发）")
            : (false, "支付处理并发冲突");
    }
}
```

### Task 6: unwrapApiEnvelope 严格化（#18）

**现状：** `miniapp/coupon-miniapp/utils/request.js:38-59` 用 `Object.keys(payload).length <= 2` 启发式判断是否为信封；合法业务数据若字段少于等于 2 会被误解包。

**方案：** 严格按 `{ code, message, data }` 三字段判定：只有同时出现 `code` 与 `message`（或 `code` 与 `data`）时才视为信封，否则原样返回。

**Files:**
- Modify: `miniapp/coupon-miniapp/utils/request.js:38-59`
- Also check: `frontend/src/utils/request.{ts,js}`（若存在相似逻辑，一并修改）

- [ ] **Step 1: 调整实现**

```javascript
function unwrapApiEnvelope(payload) {
  if (!payload || typeof payload !== 'object') return payload;

  const hasCode = Object.prototype.hasOwnProperty.call(payload, 'code');
  const hasMessage = Object.prototype.hasOwnProperty.call(payload, 'message');
  const hasData = Object.prototype.hasOwnProperty.call(payload, 'data');

  // 只有明确符合信封签名时才解包
  const isEnvelope = hasCode && (hasMessage || hasData);
  if (!isEnvelope) return payload;

  if (Number(payload.code) === 200) return payload.data;

  throw createRequestError(payload.message || '接口返回失败', {
    code: payload.code,
    payload
  });
}
```

```csharp
            entry.AbsoluteExpirationRelativeToNow = TimeSpan.FromSeconds(60);
            // 原 4 次查询合并为 1 次 JOIN
            return await db.AdminUsers.Where(u => u.Username == username && u.IsEnabled)
                .Join(db.AdminUserRoles, u => u.Id, ur => ur.AdminUserId, (u, ur) => ur.AdminRoleId)
                .Join(db.AdminRoles.Where(r => r.IsEnabled), ur => ur, r => r.Id, (_, r) => r.Id)
                .Join(db.AdminRoleMenus, r => r, rm => rm.AdminRoleId, (_, rm) => rm.AdminMenuId)
                .Join(db.AdminMenus.Where(m => m.IsEnabled), rm => rm, m => m.Id, (_, m) => m.Path)
                .Distinct().ToListAsync();
        });
    }

    public void Invalidate(string username) => cache.Remove($"admin-menus:{username}");
}
```
